# 你画我猜小游戏

一个开箱即玩的本地版「你画我猜」小游戏，无需安装任何依赖，直接打开网页即可和伙伴们一起玩。

## 功能特性

- 🎨 顺滑的画板体验，支持多种颜色、画笔粗细与橡皮擦
- 🔀 丰富的中文词库，一键抽取题目
- ⏱️ 可自定义倒计时，并支持开始、暂停与重置
- 🤔 内置猜词输入框，自动判定答案是否正确
- 💾 支持将画作导出为 PNG 图片保存
- 📱 自适应布局，桌面与移动设备均可舒适使用

## 使用方式

1. 克隆或下载本仓库代码。
2. 双击打开根目录下的 `index.html`，或通过浏览器访问该文件。
3. 点击页面右上角的「开始新回合」按钮即可开始游戏。

> TIP：多人游戏时，可让画手独自查看题目，再向其他玩家展示画板。

## 开发说明

如需本地起一个静态服务器（可选）：

```bash
cd /path/to/project
python3 -m http.server 8000
```

随后在浏览器访问 `http://localhost:8000` 即可。
